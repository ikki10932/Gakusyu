# Business Mail Editor Skill

このスキルは、取引先へのメール文章をプロフェッショナルかつ読みやすい形式に校正することを目的としています。

## 機能

以下の基準に基づいて、提供されたメール文章を校正します。

1.  **敬語の適切な使用**: 尊敬語、謙譲語、丁寧語が適切に使用されているかを確認し、必要に応じて修正します。
2.  **ビジネスメールとしての構成**: 件名、宛名、挨拶、本文、結び、署名といったビジネスメールの基本的な構成が守られているかを確認し、不足があれば補完を提案します。
3.  **明確かつ簡潔な表現**: 冗長な表現を避け、意図が明確に伝わるように簡潔な文章に修正します。
4.  **誤字脱字の修正**: 基本的な誤字脱字、文法ミスを修正します。
5.  **句読点の適切な使用**: 句読点（、。）が適切に使用されているかを確認し、読みやすさを向上させます。
6.  **ポジティブな表現**: ネガティブな表現を避け、可能な限りポジティブな言葉遣いに変換します。
7.  **末尾の定型句**: 校正後の文章の最後に必ず「以上、よろしくお願いします」を付与します。
8.  **半角スペースの排除**: 単語の前後に半角スペースを入れません。

## 使用方法

校正したいメール文章をこのスキルに提供してください。スキルは上記の基準に基づいて文章を校正し、結果を返します。

**例:**

```
ユーザー入力:
「お世話になります。先日の件ですが、まだ終わってません。急いでください。よろしく。」

スキル出力:
「お世話になっております。先日の件につきまして、現在対応中でございます。お急ぎのところ恐縮ですが、今しばらくお待ちいただけますでしょうか。以上、よろしくお願いします。」
```

## AIへの指示

このスキルを使用する際は、以下の点を厳守してください。

*   **最終的な定型句**: 校正が完了した文章の末尾には、いかなる場合でも必ず「以上、よろしくお願いします」を付与してください。この定型句は、他のいかなる文言よりも優先されます。
*   **半角スペースの排除**: 日本語の文章において、単語と単語の間、または句読点の前後など、いかなる箇所においても半角スペースを挿入しないでください。全角スペースも原則として使用せず、自然な日本語の連結を維持してください。
*   **敬語の徹底**: 取引先へのメールとして、最も適切な敬語表現（尊敬語、謙譲語、丁寧語）を選択し、丁寧すぎず不自然な箇所がないか細心の注意を払って修正してください。
*   **簡潔性**: 冗長な表現や重複する内容は排除し、伝えたい情報が明確かつ簡潔に伝わるように文章を再構築してください。
*   **ポジティブなトーン**: ネガティブな印象を与える表現は、可能な限りポジティブまたは中立的な表現に変換してください。

## 制約事項

*   本文は結論を先に書くこと。
*   提供される文章は日本語である必要があります。
*   メールの意図や背景が不明な場合、一般的なビジネス慣習に基づいた修正を行います。
*   専門用語や固有名詞の正確性については、ユーザー側で最終確認をお願いします。
*   文章は一行につき、最大全角80文字とし、それ以上の場合は区切りの良い箇所で改行してください。
*   以下の文言は変換してください。
    ・お願い申し上げます→お願いします
    ・お願いいたします→お願いします
    ・ございません→ありません
    ・いたしました→しました
    ・事象→現象
