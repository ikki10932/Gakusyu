# Document Analyzer Pro

このスキルは、ユーザーから提供された資料を分析し、その内容を明確かつ効果的に伝えるための提案書を作成します。提案書は、以下の指定された構造に従ってまとめられます。

## 使用方法

このスキルを呼び出す際は、解析対象の資料を明確に指定し、必要に応じて補足情報を提供してください。スキルは資料の内容を理解し、以下のセフォーマットで提案書を生成します。

## 提案書の出力フォーマット

提案書は、以下の5つのセクションで構成されます。各セクションは、資料の内容に基づいて詳細に記述されます。

### 1. 背景と検証目的

*   **背景**: 資料が作成された経緯や、その資料が解決しようとしている課題について記述します。
*   **検証目的**: 資料の分析を通じて達成しようとする具体的な目標や、その資料の有効性をどのように評価するかを説明します。

### 2. 検証アプローチ

*   資料の分析に用いる方法論や手順を詳述します。これには、使用するツール、データ収集方法、分析の枠組みなどが含まれます。

### 3. 期待される効果

*   提案された改善策や分析結果が、資料の理解度向上、意思決定の迅速化、または特定のビジネス目標達成にどのように貢献するかを具体的に記述します。

### 4. 主な結果

*   資料分析から得られた主要な知見、発見、または結論を簡潔にまとめます。具体的なデータや事例を引用して、結果の信頼性を高めます。

### 5. 今後の方針

*   分析結果に基づき、次に取るべき行動、推奨されるステップ、またはさらなる調査が必要な領域について提案します。短期および長期的な視点での計画を含めます。

## 例

ユーザーが「このマーケティングレポートを分析し、内容を分かりやすくするための提案を作成してください」と依頼した場合、このスキルは上記のフォーマットに従って提案書を生成します。

---

**注意**: このスキルは、提供された資料の内容を最大限に活用し、最も効果的な提案を生成することを目指します。資料が不明瞭な場合や情報が不足している場合は、追加情報の提供を求めることがあります。
